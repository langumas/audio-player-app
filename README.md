## Instalación y Ejecución

Sigue estos sencillos pasos para tener una copia local en ejecución.

### Prerrequisitos

Asegúrate de tener instalado:

* **Node.js** (versión recomendada 18+)
* **npm** (o Yarn)

### Pasos de Instalación

1.  **Clonar el repositorio:**
    ```bash
    git clone [https://github.com/](https://github.com/)<Tu-Usuario>/audio-player-app.git
    cd audio-player-app
    ```

2.  **Instalar dependencias:**
    ```bash
    npm install
    # o si usas yarn:
    # yarn install
    ```

3.  **Iniciar la aplicación en modo desarrollo:**
    ```bash
    npm start
    ```


Al pulsar MENU eliges tu MP3 que ya has descargado previamente y le das al boton de reproducción.
